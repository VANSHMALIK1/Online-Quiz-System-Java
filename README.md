# Online Quiz System (Java)

## 📌 Project Overview
The Online Quiz System is a console-based Java application designed to conduct multiple-choice quizzes and evaluate user performance dynamically. This project demonstrates the use of Object-Oriented Programming (OOP) concepts, collections framework (ArrayList), and logical score calculation in Java.

## 🚀 Features
- Multiple-choice questions (MCQs)
- Four options per question
- Instant answer validation
- Automatic score calculation
- Final result display (Pass/Fail)
- Structured and clean console interface

## 🛠 Technologies Used
- Java
- OOP (Classes & Objects)
- ArrayList (Java Collections Framework)
- Scanner (User Input Handling)

## 🗂 Project Structure
- `OnlineQuizSystem.java` – Main file containing:
  - Question class (model)
  - Quiz logic
  - Score evaluation system
